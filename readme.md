# Django-blog
我的第二个使用django写的blog程序。**填坑中**

## 环境
python3.6+Django2

## 特点
- 后台支持上传图片
- 前端采用editor.md解析markdown文档
- 添加图片灯箱

## APP
|APP名|APP用途|URL地址|
|:---:|:----:|:---:|
|home|主页|http://localhost/|
|blog|博客模块|http://localhost/blog/|

## url配置
自行配置各个模块的`urls.py`以及项目中`urls.py`
